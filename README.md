# URL Shortener Service

A simple backend service that converts long URLs into short ones
and redirects users to the original link.

## Features
- Create short URL from long URL
- Redirect by short URL
- Store links in database
- Click counter (future)

## Tech Stack
- Python
- Django
- SQLite (for development)
- Django ORM
